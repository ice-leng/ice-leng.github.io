如何在github上创建自己的博客(上)

本人刚刚在github搭建了一个博客，网上有很多教材，我主要是想写点，我在搭建过程中遇到坑。
希望我分享的能让读者少走一些弯路。

言归正传：
首先，您得有github的账号，如果没有去申请传送门

注意：请关注下你申请或者已创建的username(用户名)。


接着，登陆github账号，并创建一个仓库，仓库名称为 username + .github.io。
进入仓库，点击Settings(设置)

点击 Launch automatic page generator 创建 github page 页面

接着选择一个布局，生成布局文件。
那么恭喜您，github搭建了一个博客第一本完成了
那么去访问你的域名看看是否生成 username + .github.io
是不是很简单。
